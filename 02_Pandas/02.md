Pandas `merge()` combines DataFrames like SQL JOINs, using common keys to link employee and department data from our previous
examples.

## Basic Inner Merge

Join on matching keys only.

```python
import pandas as pd

# Employees DataFrame
employees = pd.DataFrame({
    'Name': ['Alice', 'Bob', 'Charlie', 'Diana'],
    'DeptID': ['D001', 'D002', 'D001', 'D003'],
    'Salary': [50000, 60000, 55000, 70000]
})

# Departments DataFrame
departments = pd.DataFrame({
    'DeptID': ['D001', 'D002', 'D004'],
    'DeptName': ['HR', 'IT', 'Finance']
})

# Inner merge (default) - only matching DeptID
merged = pd.merge(employees, departments, on='DeptID')
print(merged)
```

Output:

```
      Name  Salary DeptID DeptName
0    Alice   50000   D001        HR
2  Charlie   55000   D001        HR
1      Bob   60000   D002        IT
```

## Different Join Types

Control which rows appear using `how` parameter.

```python
# Left join - all employees + matching departments
left_merged = pd.merge(employees, departments, on='DeptID', how='left')

# Right join - all departments + matching employees
right_merged = pd.merge(employees, departments, on='DeptID', how='right')

# Outer join - all rows from both
outer_merged = pd.merge(employees, departments, on='DeptID', how='outer')
```

## Multiple Keys & Suffixes

Handle different column names or conflicts.

```python
# Different key names
emps = pd.DataFrame({'EmpID': [1, 2], 'Name': ['Eve', 'Frank']})
depts = pd.DataFrame({'EmployeeID': [1, 3], 'Role': ['Manager', 'Analyst']})
multi_key = pd.merge(emps, depts, left_on='EmpID', right_on='EmployeeID')

# Column name conflicts (add suffixes)
merged_suffix = pd.merge(employees, employees, on='DeptID',
                        suffixes=('_left', '_right'))
```

## Merge Types Comparison

| Type  | `how=`    | Keeps                                                                                    |
| ----- | --------- | ---------------------------------------------------------------------------------------- |
| Inner | `'inner'` | Matching rows only [w3schools](https://www.w3schools.com/python/pandas/ref_df_merge.asp) |
| Left  | `'left'`  | All left DF rows                                                                         |
| Right | `'right'` | All right DF rows                                                                        |
| Outer | `'outer'` | All rows from both                                                                       |

Use `on='key'` for same-named columns, `left_on/right_on` for different names. Perfect for joining our binned salary data
with departments. [w3schools](https://www.w3schools.com/python/pandas/ref_df_merge.asp)
