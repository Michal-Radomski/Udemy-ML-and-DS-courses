Pandas provides built-in methods for quick descriptive statistics on DataFrames, perfect for Jupyter Notebook exploration.
These summarize central tendency, dispersion, and shape using our ongoing sample data.

## Basic `describe()` Overview

Generate summary stats for numeric columns automatically.

```python
import pandas as pd

# Sample DataFrame
data = {'Name': ['Alice', 'Bob', 'Charlie'],
        'Age': [25, 30, 35],
        'Salary': [50000, 60000, 70000]}
df = pd.DataFrame(data)

print(df.describe())
```

Output:

```
              Age         Salary
count   3.000000        3.000000
mean   30.000000    60000.000000
std     5.000000    10000.000000
min    25.000000    50000.000000
25%    27.500000    55000.000000
50%    30.000000    60000.000000
75%    32.500000    65000.000000
max    35.000000    70000.000000
```

This shows count, mean, std, quartiles, min/max for numbers; categoricals get count, unique, top, freq.
[docs.vultr](https://docs.vultr.com/python/third-party/pandas/DataFrame/describe)

## Single Column Stats

Target specific columns or types.

```python
# Single column
print(df['Age'].describe())

# Numeric only, custom percentiles
print(df.describe(percentiles=[0.1, 0.9]))

# Include categoricals
print(df.describe(include='all'))
```

## Common Functions

Access individual statistics easily.

```python
print("Mean Age:", df['Age'].mean())
print("Median Salary:", df['Salary'].median())
print("Std Dev Age:", df['Age'].std())
print("Correlation:", df['Age'].corr(df['Salary']))
```

| Statistic    | Method                      | Purpose                                                                                                             |
| ------------ | --------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| Mean         | `df.mean()`                 | Average value [youtube](https://www.youtube.com/watch?v=bdygwr2DlnE)                                                |
| Median       | `df.median()`               | Middle value                                                                                                        |
| Mode         | `df.mode()`                 | Most frequent                                                                                                       |
| Variance/Std | `df.var()`, `df.std()`      | Spread measure                                                                                                      |
| Quantiles    | `df.quantile([0.25, 0.75])` | Distribution cutoffs [geeksforgeeks](https://www.geeksforgeeks.org/pandas/python-pandas-dataframe-describe-method/) |

Chain with groupby for segmented stats: `df.groupby('group_col').describe()`.
[geeksforgeeks](https://www.geeksforgeeks.org/python/how-to-get-the-descriptive-statistics-for-pandas-dataframe/)
