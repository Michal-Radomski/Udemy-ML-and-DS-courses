The `bins` parameter in Matplotlib's `plt.hist()` (or `ax.hist()`) controls how data is divided into intervals (bins) for
histogram visualization. It determines the grouping of values along the x-axis and the number/width of bars shown.

## Parameter Options

- **Integer**: Specifies the exact number of equal-width bins spanning the data range. Example: `bins=10` creates 10 bins.
- **List/Array**: Defines explicit bin edges. Example: `bins=[0, 2, 4, 6, 8]` creates bins [0-2), [2-4), etc.
- **String**: Auto-algorithms like `'auto'`, `'fd'` (Freedman-Diaconis), `'sturges'`, or `'rice'` compute optimal bins.
- **Bin width via `range()`**: `bins=range(0, 100, 10)` makes 10-unit wide bins.
  [apxml](https://apxml.com/courses/data-visualization-matplotlib-seaborn/chapter-3-basic-matplotlib-plot-types/matplotlib-histogram-bins)

## Example Usage

```
import matplotlib.pyplot as plt
import numpy as np

data = np.random.randn(1000)

plt.hist(data, bins=30)      # 30 equal bins
plt.hist(data, bins=[-3,-2,-1,0,1,2,3])  # Custom edges
plt.show()
```

Fewer bins smooth the distribution; more reveal fine details. Matplotlib defaults to `'auto'` for reasonable choices.
