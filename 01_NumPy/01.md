**NumPy** is the fundamental Python library for numerical computing, providing fast N-dimensional arrays (`ndarray`) and
mathematical functions. **Pandas** builds on NumPy for data analysis, offering `DataFrame` (Excel-like tables) and `Series`
(labeled columns) with easy data manipulation.

## NumPy Example

Fast array operations and math:

```python
import numpy as np

# Create array
arr = np.array([1, 2, 3, 4, 5])
print("Array:", arr)

# Vectorized operations (much faster than loops)
print("Squared:", arr ** 2)
print("Mean:", np.mean(arr))
print("Random 3x3 matrix:\n", np.random.rand(3, 3))

# Matrix multiplication
a = np.array([[1, 2], [3, 4]])
b = np.array([[5, 6], [7, 8]])
print("Matrix product:\n", np.dot(a, b))
```

**Key strengths**: Speed, broadcasting, linear algebra, memory efficiency.
[flexiple](https://flexiple.com/python/pandas-vs-numpy)

## Pandas Example

Tabular data manipulation:

```python
import pandas as pd
import numpy as np

# Create DataFrame
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'Salary': [50000, 60000, 70000]
}
df = pd.DataFrame(data)
print("DataFrame:\n", df)

# Easy filtering and grouping
print("Salary > 55k:\n", df[df['Salary'] > 55000])
print("Average age:", df['Age'].mean())

# Group by (simulates SQL)
df['AgeGroup'] = pd.cut(df['Age'], bins=2)
print("\nGrouped by age:\n", df.groupby('AgeGroup')['Salary'].mean())
```

**Key strengths**: Labeled data, missing values, merging, time series.
[flexiple](https://flexiple.com/python/pandas-vs-numpy)

## Quick Comparison

| Feature        | NumPy             | Pandas                                                                                                      |
| -------------- | ----------------- | ----------------------------------------------------------------------------------------------------------- |
| Data structure | Arrays            | DataFrames/Series                                                                                           |
| Best for       | Math computations | Data analysis                                                                                               |
| Speed          | Faster (pure C)   | Slower but more features                                                                                    |
| Memory         | Very efficient    | Higher usage                                                                                                |
| Indexing       | Position only     | Labels + position [geeksforgeeks](https://www.geeksforgeeks.org/python/difference-between-pandas-vs-numpy/) |

**Use NumPy** for heavy math/ML preprocessing, **Pandas** for data cleaning/exploration.
