**NumPy alternative in JavaScript/TypeScript**: **NumJs** (`nj`) or **TensorFlow.js** tensors  
**Pandas alternative**: **Danfo.js** (closest Pandas-like API) or **Data-Forge**

## NumJs (NumPy equivalent)

Array operations and linear algebra:

```javascript
// npm install numjs
const nj = require("numjs");

// Create array
let arr = nj.array([1, 2, 3, 4, 5]);
console.log("Array:", arr.tolist());

// Vectorized math
console.log("Squared:", arr.pow(2).tolist());
console.log("Mean:", arr.mean());

// Matrix operations
let a = nj.array([
  [1, 2],
  [3, 4],
]);
let b = nj.array([
  [5, 6],
  [7, 8],
]);
console.log("Dot product:", nj.dot(a, b).tolist());
```

## Danfo.js (Pandas equivalent)

DataFrames with familiar syntax:

```javascript
// npm install danfojs-node
const dfd = require("danfojs-node");

const data = {
  Name: ["Alice", "Bob", "Charlie"],
  Age: [25, 30, 35],
  Salary: [50000, 60000, 70000],
};

const df = new dfd.DataFrame(data);
console.log(df.head());

// Filtering
const highSalary = df.query({ column: "Salary", is: ">", to: 55000 });
console.log(highSalary);

// Group by
const grouped = df.groupby(["Age"]).mean();
console.log(grouped);
```

## Data-Forge (Alternative Pandas)

CSV handling and transformations:

```javascript
// npm install data-forge
const dataForge = require("data-forge");

const csvData = `Name,Age,Salary
Alice,25,50000
Bob,30,60000
Charlie,35,70000`;

const df = dataForge.fromCSV(csvData).parseInts(["Age", "Salary"]);

console.log(df.toArray());
```

## Quick Comparison

| Purpose      | Python | JavaScript/TS            |
| ------------ | ------ | ------------------------ |
| Arrays/Math  | NumPy  | **NumJs**, TensorFlow.js |
| DataFrames   | Pandas | **Danfo.js**, Data-Forge |
| Browser/Node | Both   | **All support both**     |

**Danfo.js** has the most Pandas-like syntax. **NumJs** matches NumPy operations best.
