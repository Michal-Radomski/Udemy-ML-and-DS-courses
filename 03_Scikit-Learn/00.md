Scikit-learn (sklearn) is Python's premier open-source machine learning library for classification, regression, clustering,
and preprocessing. It provides consistent APIs built on NumPy/SciPy, making model experimentation fast.
[codecademy](https://www.codecademy.com/article/scikit-learn-tutorial)

## Simple Classification Example

Predict iris flower species using our Pandas workflow data.

```python
import pandas as pd
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# Load data (works like our employee DataFrames)
iris = load_iris()
df = pd.DataFrame(iris.data, columns=iris.feature_names)
df['species'] = iris.target

# Prep (like our merge/cut examples)
X = df.drop('species', axis=1)
y = df['species']
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Train model (sklearn's consistent pattern)
model = RandomForestClassifier(n_estimators=100)
model.fit(X_train, y_train)

# Predict & evaluate
predictions = model.predict(X_test)
print("Accuracy:", accuracy_score(y_test, predictions))
```

This outputs ~95-100% accuracy, showing sklearn's power with minimal code.

## JavaScript/TypeScript Equivalents

| Library           | Focus                        | Key Features                                                        | Install                                                                                     |

| **ml.js**         | Classical ML                 | Linear regression, SVM, clustering, decision trees                  | `npm i ml`                                                                                  |
| **brain.js**      | Neural Networks              | Simple NNs for JS/Node/browser                                      | `npm i brain.js`                                                                            |
| **Danfo.js**      | DataFrames + ML              | Pandas-like DataFrames + basic ML                                   | `npm i danfojs-node` [codecademy](https://www.codecademy.com/article/scikit-learn-tutorial) |
